# Thinking of Who

### Ever want to play your own custom-made classic guessing game game? Look no further!

# Creating a Game
![create_page](https://github.com/user-attachments/assets/c1c8036f-c464-4dd3-a548-29a2c3a35a14)

Create a game with up to 24 images (You can add more later).
Upload the images from your computer's local files or paste them directly from your clipboard.
Create a name for the game you made so you can search it up later or in case you lose track of the code. If you don't want to allow others to be able search and find the game you made, uncheck the 'Searchable' box.

# Playing a Game
![play_game](https://github.com/user-attachments/assets/7136bb87-840e-499a-a1b2-69632e08392b)


### How to play (2 players):
Your goal is deduce which thing/person/place your opponent chose by taking turns asking yes or no questions. Play in 3-5 rounds. Don't reuse questions.
